## Youdo implemented with Node.js 

Web app to find tasks and post jobs 

**Dependencies** 

* node.js > 0.10 
* mysql/mariadb 

Nice to have [PM2](https://github.com/Unitech/pm2) process manager for node 

## Installation 

```
git clone https://github.com/artsoroka/youdo && cd youdo 

npm install 

node app/app.js 

```
